
Many Icons, and other assets are designed in 4x. 4 times the size they are used in. This method helps the designer create more details and crisp assets. If you need to make changes to the supplied assets, you need to first run the "create-icon.atn" Photoshop action, before you save them for use. This action shrinks the asset to the size it needs to be for use in the Connector. 

To load this actions into the Actions palette.

- Choose Load Actions from the Actions palette menu. Locate and select the action set file, and then click Load. Select an action set from the bottom of the Actions palette menu.
